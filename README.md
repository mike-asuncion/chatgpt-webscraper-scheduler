# ChatGPT Web Scraper Scheduler

A WordPress plugin that automates web content scraping, AI-powered rewriting, and scheduled publishing to your site.

## Overview

ChatGPT Web Scraper Scheduler lets you add URLs from sites you want to source content from. The plugin scrapes the content, rewrites it using ChatGPT (OpenAI), and automatically publishes the result as a new post on your WordPress site. You can also set up schedules to run these tasks automatically at defined intervals.

## Features

- Add and manage source URLs to scrape content from
- AI-powered content rewriting using ChatGPT (OpenAI API)
- Auto-publish rewritten content as WordPress posts
- Schedule scraping and posting tasks (daily, weekly, custom intervals)
- Simple admin interface inside the WordPress dashboard

## Requirements

- WordPress 5.0 or higher
- PHP 7.4 or higher
- OpenAI API key

## Installation

1. Download or clone this repository into your `/wp-content/plugins/` directory.
2. Activate the plugin through the **Plugins** menu in WordPress.
3. Go to the plugin settings page and enter your **OpenAI API key**.
4. Add the URLs you want to scrape and configure your schedule.

## Usage

1. Navigate to the plugin menu in your WordPress admin dashboard.
2. Add the URLs you want to scrape content from.
3. Configure how you want the content to be rewritten and posted.
4. Set a schedule (optional) to automate the process.
5. Run manually or let the scheduler handle it automatically.
